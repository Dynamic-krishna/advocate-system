# Advocate Registration System

A simple web-based registration and login system for advocates built with PHP and PostgreSQL.

## Features

- User registration with validation
- Secure login system
- Password hashing for security
- Data encryption for sensitive information (mobile, email)
- User dashboard with profile management
- Photo upload functionality
- Session management

## Technologies Used

- PHP 8.2+
- PostgreSQL
- HTML/CSS
- Docker (for deployment)

## Local Installation

### Prerequisites
